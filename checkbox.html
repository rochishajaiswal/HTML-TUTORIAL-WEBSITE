Checkbox Controls

Checkboxes are used when more than one option is required to be selected. They are also created using HTML <input> tag but type attribute is set to checkbox.

Example

Code Snippet :

<form>
<input type = "checkbox" name = "maths" value = "on"> Maths
<input type = "checkbox" name = "physics" value = "on"> Physics
</form>

Preview :

Maths Physics

Attributes

Sl. Attribute Description Values
1 type Indicates the type of input control and for checkbox input control it will be set to checkbox. checkbox.
2 name Used to give a name to the control which is sent to the server to be recognized and get the value. User decided.
3 value The value that will be used if the checkbox is selected. User decided.
4 checked Set to checked if you want to select it by default. No value.
