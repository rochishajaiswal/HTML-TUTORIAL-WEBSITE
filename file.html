File Upload Box

If you want to allow a user to upload a file to your web site, you will need to use a file upload box, also known as a file select box. This is also created using the <input> element but type attribute is set to file.

Example

Code Snippet :

<form>
<input type = "file" name = "fileupload" accept = "image/*" />
</form>

Preview :

Attributes

Sl. Attribute Description Values
1 name Used to give a name to the control which is sent to the server to be recognized and get the value. User decided.
2 accept Specifies the types of files that the server accepts. audio/*, image/*, video/*, etc.
