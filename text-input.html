Text Input Controls

There are three types of text input used on forms -
  • Single-line text input controls
  • Password input controls
  • Multi-line text input controls

1. Single Line text input

This control is used for items that require only one line of user input, such as search boxes or names. They are created using HTML <input> tag.

Example

Code Snippet :

<form >
First name: <input type = "text" name = "first_name" />
<br>
Last name: <input type = "text" name = "last_name" />
</form>

2. Password input controls

This is also a single-line text input but it masks the character as soon as a user enters it. They are also created using HTMl <input> tag but type is set to password.

Example

Code Snippet :

<form >
User ID : <input type = "text" name = "user_id" />
<br>
Password: <input type = "password" name = "password" />
</form>

Preview :

User ID :
Password:

Attributes

Sl. Attribute Description Values
1 type Indicates the type of input control. email,password,tel,url,etc.
2 name Used to give a name to the control which is sent to the server to be recognized and get the value. User decided.
3 value This can be used to provide an initial value inside the control. User decided.
4 size Allows to specify the width of the text-input control in terms of characters. User decided.
5 maxlength Allows to specify the maximum number of characters a user can enter into the text box. User Decided.

3. Multiple-Line Text Input Controls

This is used when the user is required to give details that may be longer than a single sentence. Multi-line input controls are created using HTML <textarea> tag.

Example

Code Snippet :

<form>
Description : </br>
<textarea rows = "5" cols = "50" name = "description">
Enter description here...
</textarea>
</form>

Attributes

Sl. Attribute Description Values
1 name Used to give a name to the control which is sent to the server to be recognized and get the value. User decided.
2 rows Indicates the number of rows of text area box. User decided.
3 cols Indicates the number of columns of text area box User decided.
