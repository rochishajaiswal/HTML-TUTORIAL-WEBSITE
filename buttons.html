Button Controls

There are various ways in HTML to create clickable buttons. You can also create a clickable button using <input>tag by setting its type attribute to button.

Example

Code Snippet :

<form>
<p>Type something in the input box and press reset button to see how it works</p>
<input type = "text" value="Clicking the reset value will delete this line"/><br>
<input type = "submit" name = "submit" value = "Submit" />
<input type = "reset" name = "reset" value = "Reset" />
<input type = "button" name = "ok" value = "OK" />
</form>
