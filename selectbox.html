Select Box Controls

A select box, also called drop down box which provides option to list down various options in the form of drop down list, from where a user can select one or more options.

Example

Code Snippet :

<form>
<select name = "dropdown">
<option value = "Maths" selected>Maths</option>
<option value = "Physics">Physics</option>
</select>
</form>

Preview :

Attributes

Following is the list of important attributes of <select> tag -
Sl. Attribute Description Values
1 name Used to give a name to the control which is sent to the server to be recognized and get the value. User decided.
2 size This can be used to present a scrolling list box. User decided.
4 mulitple If set to "multiple" then allows a user to select multiple items from the menu. No value.
Following is the list of important attributes of <option> tag -
Sl. Attribute Description Values
1 value The value that will be used if an option in the select box box is selected. User Decided.
2 selected Specifies that this option should be the initially selected value when the page loads. No value.
3 label An alternative way of labeling options User decided.
